If you are handwriting the scientific name for the milkweed, Asclepias incarnata, how is it different from typing the name?

(A) ** The scientific name is underlined when handwritten.

(B) The 2nd part of the scientific name, the species, is capitalized, when handwritten.

(C) The 1st part of the scientific name, the genus, is lowercase, when handwritten.

(D) Both parts of the scientific name are lowercase, when handwritten.

EXPLANATIONS BELOW

Concept note-1: -If you are handwriting the scientific name for the milkweed, Asclepias incarnata, how is it different from typing the name? The scientific name is underlined when handwritten. The 2nd part of the scientific name, the species, is capitalized, when handwritten.

Concept note-2: -There are rules to follow when writing a scientific name. The genus name is written first. The genus name is always underlined or italicized. The first letter of the genus name is always capitalized.

Concept note-3: -Scientific names are always italicized. When hand-writing text, you can underline them instead, but there is no excuse not to italicize when using a word processor. Example: use Bos taurus, not Bos taurus. 2.

Concept note-4: -Binomial nomenclature gives each species a scientific name. Scientific name consists of two parts. The first is genus name and second is species name. It is always written in italics (if typed) or underlined (handwritten).

Concept note-5: -The binomial name consists of a genus name and specific epithet. The scientific names of species are italicized. The genus name is always capitalized and is written first; the specific epithet follows the genus name and is not capitalized. There is no exception to this.
