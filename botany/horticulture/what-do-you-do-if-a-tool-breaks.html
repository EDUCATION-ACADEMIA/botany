What do you do if a tool breaks?

(A) ** Tell the teacher

(B) Hide it somewhere

(C) Stop using it

(D) Throw it at someone

EXPLANATIONS BELOW

Concept note-1: -Provide timely feedback so mistakes can be responded to The learning process usually follows these steps in this order: practice activities, make errors, get feedback, think over the feedback, and try again. The less this process is interrupted the more efficient and effective the learning is.

Concept note-2: -A teacher tool is any tool that enhances or expedites the teaching process and helps teachers to teach better. For instance, it can be a tool that allows the teachers to conduct classes in a more organized and efficient way and communicate with parents quickly and efficiently to improve classroom productivity.
